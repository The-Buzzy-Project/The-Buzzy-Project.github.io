Golden Ticket is a free and open source launcher for Sim Theme Park on modern systems!

Get It Now!Learn More

Any Windows Version*

Yes, any! As long as you're running a fairly modern version of Windows, you can get the game running!
Both 32-bit and 64-bit are supported!

* Any Windows version from Vista to Windows 10

Tweaked for stability

Golden Ticket includes modified configuration files for the game which help it run smoother! In some cases, these tweaks allow it to run at 60fps!

Open Source

Golden Ticket is open source, which means you anyone can look at the code, as well as contribute! Best of all, that means Golden Ticket is completely free!

About

Golden Ticket plans to include many features in future updates! Some of which include installing Gold Edition content, fixing missing park music and opening videos, and in the far far future, Golden Ticket will also act as a mod manager!

Fast, clean, and easy!

Golden Ticket aims to be easy to use. That means you don't need to go through a bunch of menus, make complex selections, or wait for long downloads. Just install, click, and play!

  • Simple, clean UI
  • Fast and easy to use
  • Works on most systems

How Golden Ticket works

Golden Ticket works by taking the collective knowledge of others who love the game just like you, and combines it all into one package, allowing the game to run.

    The following fixes are included...

  • Adam Hearn's font file fix
  • lecture's fix for missing text on signs
  • Lhug's fix for crashing on Windows 7+
  • Bullfrog's official 2.0 patch

It's open source!

This means the source code for it is available for anyone to view, or even contribute! That also means you can trust Golden Ticket to do what it's supposed to, and nothing more.

  • Source code is free to view
  • Community supported
  • Trustworthy
